# VFA Golden Ratio Brand System 2026

This package contains a complete VFA identity starter system designed with golden-ratio proportions for a global digital brand architecture.

## Contents

- Primary VFA logo mark in SVG and PNG
- Monochrome and transparent variants
- Golden-ratio construction guide
- Horizontal lockups
- App icons, favicons, iOS and Android icon sets
- Social image and hero background
- Pattern system
- Brand tokens and CSS
- Developer asset map
- Brand book
- Figma refinement brief

## Recommended first use

1. Open `07_WEB_ASSETS/vfa-specimen.html`
2. Review `09_DOCS/VFA_GOLDEN_RATIO_BRAND_BOOK_2026.md`
3. Use `01_LOGO/svg/vfa-golden-ratio-primary-mark.svg` as the source of truth
4. Give `10_FIGMA_BRIEF/FIGMA_REFINEMENT_BRIEF.md` to the design team for further refinement

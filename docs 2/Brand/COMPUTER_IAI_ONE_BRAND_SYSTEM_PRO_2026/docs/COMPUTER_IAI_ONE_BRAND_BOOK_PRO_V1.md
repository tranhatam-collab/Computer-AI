# COMPUTER.IAI.ONE BRAND BOOK PRO v1.0

## 1. Brand definition

Computer.iai.one is the visual identity for a Global Self-Upgrading Personal AI Computer Infrastructure.

The brand must communicate:

- intelligence without noise
- security without fear
- enterprise-grade infrastructure without coldness
- personal ownership without consumer gimmick
- IAI.ONE synchronization without visual clutter

## 2. Logo system

### Primary mark

Use `computer-iai-one-mark-primary.svg` when the mark appears alone.

### Horizontal lockup

Use `computer-iai-one-lockup-horizontal.svg` for homepage header, decks, PDF covers, and partner materials.

### Stacked lockup

Use `computer-iai-one-lockup-stacked.svg` for square or centered layouts.

### Mono marks

Use mono black or white only when color reproduction is limited.

## 3. Clear space

Minimum clear space equals one IAI core stroke width around the full logo.

## 4. Minimum sizes

- Mark: 32 px digital minimum.
- Horizontal lockup: 180 px width minimum.
- Stacked lockup: 120 px width minimum.

## 5. Color roles

Deep Navy is the base. Verification Gold is used for trust and proof. Technical Cyan is used for routing and live intelligence. Parchment is used for premium editorial surfaces.

## 6. Typography

Display: Source Serif 4 or Fraunces.
UI: Inter or system sans-serif.
Mono: JetBrains Mono or SF Mono.

## 7. UI direction

The interface should feel like a serious control surface for a private AI computer, not a toy chatbot. Use cards, status badges, traces, command surfaces, and verification evidence blocks.

## 8. Prohibited use

Do not stretch the logo. Do not place it on low contrast colors. Do not add emoji or decorative effects. Do not use random gradients outside the official palette. Do not use the mark to imply production readiness before evidence exists.

## 9. Recommended web placement

- Header: mark + Computer.iai.one.
- Hero: horizontal lockup or mark with headline.
- Footer: IAI.ONE synchronized system note.
- OG image: use provided PNG.
- Favicon: use provided ICO and PNG files.

## 10. Public wording

Approved phrase:

Global Self-Upgrading Personal AI Computer Infrastructure.

Avoid:

- AI does everything.
- Absolute security.
- 100 percent error-free.
- Production-ready without verification.

# Computer.iai.one Brand System PRO v1.0

This package contains the first production-oriented visual identity kit for Computer.iai.one, synchronized with the IAI.ONE ecosystem.

## Strategic role

Computer.iai.one is not an app, chatbot, or product catalog. It is a Global Self-Upgrading Personal AI Computer Infrastructure. Each user owns an AI Computer Instance with Super Apps, Agent Team, Data Vault, Memory, Command Kernel, Security Kernel, Verification Engine, Sync Layer, and governance.

## Visual concept

The mark combines five ideas:

1. Instance frame: the rounded square represents each private AI Computer Instance.
2. IAI core: three vertical strokes encode IAI inside the computer core.
3. Agent orbit: small nodes and arcs represent multi-agent verification.
4. Command path: the cyan A-shaped signal represents command flow and intelligence routing.
5. Trust seal: gold perimeter and node represent proof, audit, and governance.

## Main palette

- Deep Navy: primary infrastructure color.
- Parchment: premium IAI system background.
- Verification Gold: trust, proof, governance.
- Technical Cyan: AI routing, runtime, live system signals.
- Graphite: text and admin surfaces.
